**SysAnalytiq** — это демонстрационная веб-платформа, разработанная в рамках выпускной квалификационной работы. Она объединяет специализированные инструменты для обучения системному анализу в едином интерфейсе, предоставляя возможности для практики и проверки знаний.

## Возможности платформы

### 1. Встроенные редакторы (5 модулей)
Платформа предоставляет единую среду для работы с ключевыми нотациями и языками:
*   **SQL**: Редактор кода с подсветкой синтаксиса и возможностью выполнения запросов.
*   **BPMN**: Визуальный редактор для моделирования бизнес-процессов (на базе `bpmn-js`).
*   **ERD**: Визуальный редактор для проектирования схем баз данных (на базе `React Flow`).
*   **PlantUML**: Редактор "диаграммы как код" с мгновенным предпросмотром.
*   **Swagger (OpenAPI)**: Редактор спецификаций API с визуализацией в Swagger UI.

### 2. Режим "Песочница"
Свободный режим, где студенты могут практиковаться в использовании инструментов без привязки к конкретным заданиям.

### 3. Модуль обучения
*   **Выполнение заданий**: Интерфейс для решения практических задач в каждом из модулей.
*   **Автоматическая проверка**: Базовая валидация решений (например, проверка синтаксиса, наличия обязательных элементов диаграммы, сравнение результатов SQL-запросов).
*   **Ручная проверка**: Возможность отправки решений преподавателю для получения обратной связи.

### 4. Админ-панель
*   **Конструктор заданий**: Создание упражнений с настройкой критериев автоматической проверки.
*   **Проверка решений**: Интерфейс для просмотра работ студентов и выставления оценок.

## Технологический стек

### Frontend
*   **Framework**: React 18 + TypeScript
*   **Build Tool**: Vite
*   **Styling**: Tailwind CSS + Shadcn UI
*   **State Management**: React Context / Hooks
*   **Libraries**:
    *   `reactflow` (ERD диаграммы)
    *   `bpmn-js` (BPMN диаграммы)
    *   `@monaco-editor/react` (Редактор кода)
    *   `swagger-ui-react` (Визуализация API)

### Backend
*   **Language**: Python
*   **Framework**: Django Ninja
*   **Architecture**: Clean Architecture
*   **Database**: PostgreSQL / SQLite

## Установка и запуск

### Предварительные требования
*   Node.js (версия 18 или выше)
*   npm

### Запуск Frontend части

1.  Перейдите в папку фронтенда:
    ```bash
    cd frontend
    ```

2.  Установите зависимости:
    ```bash
    npm install
    ```

3.  Запустите сервер разработки:
    ```bash
    npm run dev
    ```

4.  Откройте приложение в браузере по адресу: `http://localhost:5173`
